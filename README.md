# Alkenstern VTT

Homebrew-Modul für **Foundry VTT (PF2E)** zur Verwaltung von:

- 🧭 Vorbereitungspunkten  
- ⏱️ Verstrichener Zeit  
- 🎲 Integration mit Monk's TokenBar  
- 🧰 Komfort-Makros für Auszeit-Aktivitäten  

---

## ✨ Features

### Vorbereitungspunkte

- Automatische SpecialResource pro Charakter
- Unterstützt:
  - Kritischer Erfolg: **+2 VP**
  - Erfolg: **+1 VP**
  - Kritischer Fehlschlag: **−1 VP**
- Anzeige im Effekt (einzeilig)
- Kein Token-Icon
- Spieler können Effekt nicht löschen (optional)

---

### Zeit-Tracker

- Misst verstrichene Zeit in Stunden
- Anzeige formatiert als: 2 Tage 12 h
- Wird automatisch erhöht (z. B. +12 h pro Aktivität)
- Hartes Limit möglich (Standard: 4 Tage)

---

### Monk's TokenBar Integration

Das Modul stellt eine robuste Bridge bereit:

- ✅ requestRoll Wrapper  
- ✅ zuverlässige DoS-Erkennung  
- ✅ nat20 / nat1 korrekt berücksichtigt  
- ✅ wartet sauber auf Spielerwürfe  

---

### GM-Tools

Enthaltene Makro-Bausteine ermöglichen:

- Auszeit-Aktivitäten
- Zeitbearbeitung
- VP-Übersichten
- Batch-Operationen
